El CSS se usa para dar estilo a documentos HTML. Se usan distintos tipos de selectores para seleccionar elementos y darles estilo. Por ejemplo, en este caso se uso un selector global * para cambiar la fuente en todo el documento usando la propiedad font-family y se selecciono el elemento header usando un selector de elemento para alinear sus contenidos como una fila.

Borders

Los elementos también se pueden seleccionar usando su id, poniendo un # antes de su id en el archivo CSS. Se le puede dar un border a un elemento usando la propiedad border a la cual primero se especifica el grosor del border, el estilo de border (solid, dashed, none, grooved, etc). Se le puede asignar una curvatura al border usando la propiedad border-radius a la cual se le asigna un valor númerico de su curvatura.

A los elementos tambien se les puede dividir la border por lado del elemento usando las propiedades border-left, border-right, border-bottom, border-top con el mismo formato de valores. Otra forma de seleccionar elementos en CSS es por medio de clases, las cuales se asignan en los elementos html con el atributo class y se pone un punto (.) antes del selector en el archivo CSS para indicar que es un selector de clase.

Padding

Se puede usar la propiedad padding para crear espacio dentro de un elemento. Por ejemplo, en este caso se uso la propiedad con un valor de 15px para que los elementos anidados en el div no estuvieran pegados al border azul. Esta propiedad puede recibir uno, dos o 4 valores. En el caso de que reciba uno se usa para el padding en todas las direcciones. En caso de que reciba dos el primero aplica para arriba y abajo y el segundo para los lados. En caso de que reciba 4 se usan para arriba, derecha, abajo, izquiera respectivamente.

Margen

Se usa la propiedad margin para indicar espacio entre elementos. Por ejemplo, en este caso se le aplica al elemento div para que exista una separación entre las secciones de esta página.

Background image

Se puede usar la propiedad background-image para usar una imagen como fondo de un elemento. Se le pasa el path o URL de la imagen usando la función url

Color

Se puede usar la propiedad color para cambiar el color del foreground, usando colores predeterminados, RGB o hexadecimales.

Filter

Se puede usar la propiedad filter con diferentes funciones como blur, contrast, grayscale para modificar la apariencia de una imagen.

Posicionamiento

Se puede usar la propiedad position para indicar como se debe acomodar un elemento en el documento. Se le pueden asignar los siguientes valores:

  • static: Posicionamiento normal en HTML.
  • relative: Posicionamiento relativo a su posición normal.
  • fixed: Se posiciona relativo al viewport por lo que mantiene su posición aunque el usuario mueva la página.
  • absolute: Se posiciona relativo al primer ancestro posicionado.

Fuentes y Texto

Como se menciono previamente se puede usar la propiedad font-family para cambiar la fuente usada en el documento. Las siguientes propiedades también se pueden usar para dar estilo a texto:

  • font-weight: Se usa para especificar el peso o boldness de la fuente.
  • font-style: Se usa para indicar el estilo de la fuente con valores normal, italics, oblique.
  • font: Se usa como shorthand o combinación de las propiedades anteriores, pudiendo especificar font-style, font-variant, font-weight, font-size, font-family.

Estilo de listas

Se puede usar la propiedad list-style para especificar el tipo de bullets a usar (ej. bullets, square), la posición de la lista (inside, outside) o una imagen para los marcadores de elementos. Estas propiedades se pueden especificar en cualquier orden.

Overflow

Se puede usar la propiedad overflow para indicar el comportamiento del contenido de un elemento cuando el tamaño del contenido lo excede. Puede tener los siguientes valores:

  • visible: Comportamiento predeterminado, el contenido no se corta.
  • hidden: Lo que no cabe es cortado y no se muestra.
  • scroll: El contenido se corta y se añaden "scroll bars" para ver el resto.
  • auto:: Similar a scroll pero solo se añaden "scroll bars cuadno es necesario.
