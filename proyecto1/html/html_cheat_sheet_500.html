Tags semanticos

En general, es una buena idea usar tags semanticos en una página web ya que le dan sentido al documento, tanto para los que trabajan en este como para motores de busqueda o screen readers. Además de esto pueden ser utiles para agrupar elementos y darles estilos usando CSS.

Dentro del header

El tag header se usa para el contenido de introducción de una sección, usualmente también conteniendo elementos de navegación.

Imagenes

Para insertar una imagen se usa el tag img. Este tag tiene diferentes atributos, entre ellos

  • src: Path de la imagen, ya sea un URL o un path relativo a un archivo del proyecto.
  • alt: Atributo de accesibilidad usado para describir el contenido de la imagen para aquellos con screen readers o conexiones a internet insuficientes para cargar la imagen.
  • width: Anchura de la imagen en pixeles.

Links

Para crear links se usa el elemento a junto con el atributo href que indica la dirección a la que lleva el link (puede ser path relativo a un archivo del proyecto/servidor) y el atributo target el cual se usa para especificar como abrirlo. Por ejemplo, usando _blank se abrirá en una pestaña nueva.

Formas

Se usa para recolectar información del usuario y se crean usando el elemento form. Se pueden anidar los siguientes elementos:

  • textarea: Caja de texto
  • input: Campo de texto. Se puede modificar el tipo de entrada usando el atributo type.
  • button: Botones. Usar type="submit" para un boton que envie la forma.

En las entradas de texto se puede usar el atributo placeholder para indicar una instrucción. Se pueden añadir etiquetas usando label.
